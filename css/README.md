# EduPlatform

## Описание

EduPlatform — это простой веб-проект образовательной платформы c различными курсами.  
Проект содержит главную страницу с приветственным заголовком, информационным блоком и базовой стилизацией по макету Figma.

## Структура проекта

/css - стили проекта
/js - JavaScript файлы
/assets - дополнительные ресурсы
/images - изображения
/design - ссылка на Figma-макет
index.html - главная страница
package.json - npm-пакет и зависимости
README.md - этот файл

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js (npm)
- Prettier (форматирование кода)
- Figma (дизайн и прототипирование)

## Стили

- Шрифты: Montserrat
- Основной фон: #C9F0FF
- Основные блоки: #A8C5F9
- Размер текста: 20px
- Цвет текста: #000000
